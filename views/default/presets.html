Help

Lists of alleles should be uploaded in plain text csv files with one entry per row. The column should be headed with the label 'allele'. The current limit is 50 alleles.

Naming conventions:

MHC-I allele: HLA-A*68:02

For MHC-II alleles we use the following format (without the colon separator): HLA-DRB1*0101

See http://hla.alleles.org/nomenclature
